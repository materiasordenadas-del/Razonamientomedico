# RC1 — Caso clínico respiratorio

## Contenido

- `case_data.js`: datos clínicos, manejo y evaluación experta del caso.
- `metadata.json`: metadatos públicos para biblioteca y herramientas futuras.
- `assets/`: recursos multimedia exclusivos del caso cuando estén disponibles.

## Regla de confidencialidad pedagógica

La portada solo debe usar `metadata.json` o los datos públicos equivalentes en `data/cases_catalog.js`. No debe leer ni mostrar `case_data.js` antes de iniciar el caso, porque contiene datos que revelan la respuesta diagnóstica.

## Recursos pendientes

- Radiografía de tórax PA.
- Radiografía lateral de tórax.
- Audio de estertores crepitantes focales.
- Imagen de presentación del paciente, únicamente con licencia compatible.

No usar recursos protegidos o con datos identificables de pacientes.
