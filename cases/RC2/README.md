# RC2 — Dolor torácico agudo de alto riesgo

## Contenido

- `case_data.js`: datos clínicos, manejo y evaluación experta del caso.
- `metadata.json`: metadatos públicos para biblioteca y herramientas futuras.
- `assets/`: recursos multimedia exclusivos del caso cuando estén disponibles.

## Estándar de evaluación experto

RC2 incorpora `expertEvaluation.comparisonByModule` para comparar, en cada módulo:

1. hallazgos esperados, críticos omitidos y datos de bajo valor;
2. representación esperada de la enfermedad actual;
3. Tier 3 experto: diagnóstico líder, alternativa y diagnóstico que no se puede perder;
4. pausa diagnóstica y plan de manejo esperado.

## Recursos pendientes

- ECG inicial y seriado compatibles con el caso.
- Ecocardiograma o imagen ilustrativa con licencia compatible.
- Imagen de presentación sin datos identificables.

No usar material protegido ni recursos que identifiquen pacientes.
